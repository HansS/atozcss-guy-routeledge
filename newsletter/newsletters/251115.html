Great CSS Reads

Get Started with CSS Polygons

My new project is full of triangles, polygons and 3D. It's not quite as exciting as the Species in Pieces that did the rounds earlier this year by Bryan James. The creator of that awesome side project has written up a great getting started guide to working with CSS polygons for Creative Bloq. Check it out.

Clippy: CSS clip-path maker

Continuing the theme of polygons, I came across this brilliant interactive tool for creating them and then grabbing the code required to build the shape you've created. @bennettfeely has created a great tool here and I'd highly recommend having a play with it.

How to Make Yourself Indispensible

Starting out is tough and in this modern age where design and web development are starting to be comoditised, it can sometimes feel like you have to cut your price and work twice as long to win the favour of clients and employers. There's some great words of wisdom in this piece - a great read for creative professionals of all levels.

18 Examples of SVG with Awesome Effects

SVG is great as an image format but it's also incredibly versatile when it comes to animation. It's not something I've had the opportunity to play with much yet but this round up post is certainly full of inspiring ideas for what's possible.

Material Design Ripples with CSS

In this tutorial, Ionut goes through the process of building Google's Material Design ripples in pure CSS using some pseudo classes, sibling combinator selectors and a dash of keyframe animations.
